# 桌面
## 恢复win32 disk imager刻录过的U盘

1、按`win`+`R`在运行框`diskpart`

2、在命令行输入`list disk`列出所有硬盘

3、输入`select disk 1`，可根据容量判断是1还是？

4、输入`clean`，清除磁盘；

5、输入`create partition primary`，创建分区；

6、输入`active`，激活分区；

7、输入`format fs=ntfs quick`，以ntfs格式化硬盘；

8、分配盘符。
